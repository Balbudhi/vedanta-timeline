# Upadeśa Sāram (The Essence of Instruction) — Sanskrit, 30 verses
# Author: Ramaṇa Maharṣi (Bhagavān Śrī Ramaṇa Maharṣi), composed 1927.
# Language: Sanskrit (IAST). Original verse text only; English translation/commentary omitted.
# Source: Arunachala Ashrama, https://arunachala.org/docs/upadesa-saram/

1. kartur ājñayā prāpyate phalaṃ / karma kiṃ paraṃ karma tajjaḍam
2. kṛti-maho-dadhau patana-kāraṇam / phalam aśaśvataṃ gati-nirodhakam
3. īśvarārpitaṃ necchayā kṛtam / citta-śodhakaṃ mukti-sādhakam
4. kāya-vāṅ-manaḥ kāryam uttamam / pūjanaṃ japaś cintanaṃ kramāt
5. jagata īśadhī-yukta-sevanam / aṣṭa-mūrti-bhṛd-deva-pūjanam
6. uttama-stavād ucca-mandataḥ / cittajaṃ japa-dhyānam uttamam
7. ājya-dhārayā srotasā samam / sarala-cintanaṃ viralataḥ param
8. bheda-bhāvanāt so'ham ity asau / bhāvanābhidā pāvanī matā
9. bhāva-śūnya-sad-bhāva-susthitiḥ / bhāvanā-balād bhaktir uttamā
10. hṛt-sthale manaḥ svasthatā kriyā / bhakti-yoga-bodhāś ca niścitam
11. vāyu-rodhanāl līyate manaḥ / jāla-pakṣivad rodha-sādhanam
12. citta-vāyavaś cit-kriyā-yutāḥ / śākhayor dvayī śakti-mūlakā
13. laya-vinaśane ubhaya-rodhane / laya-gataṃ punar bhavati no mṛtam
14. prāṇa-bandhanāl līna-mānasam / eka-cintanān nāśam ety adaḥ
15. naṣṭa-manaso 'tkṛṣṭa-yoginaḥ / kṛtyam asti kiṃ svasthitiṃ yataḥ
16. dṛśya-vāritaṃ cittam ātmanaḥ / cittva-darśanaṃ tattva-darśanam
17. mānasaṃ tu kiṃ mārgaṇe kṛte / naiva mānasaṃ mārga ārjavāt
18. vṛttayas tv ahaṃ-vṛttim āśritāḥ / vṛttayo mano viddhy ahaṃ manaḥ
19. aham ayaṃ kuto bhavati cinvataḥ / ayi pataty ahaṃ nija-vicāraṇam
20. ahami nāśa-bhāji aham-ahaṃtayā / sphurati hṛt svayaṃ parama-pūrṇa-sat
21. idam ahaṃ padābhikhyam anvaham / aham-i līnake 'py alaya-sattayā
22. vigrahendriya-prāṇa-dhī-tamaḥ / nāham eka-sat taj jaḍaṃ hy asat
23. sattva-bhāsikā cit kva vetarā / sattayā hi cic cittayā hy aham
24. īśa-jīvayor veṣa-dhī-bhidā / sat-svabhāvato vastu kevalam
25. veṣa-hānataḥ svātma-darśanam / īśa-darśanaṃ svātma-rūpataḥ
26. ātma-saṃsthitiḥ svātma-darśanam / ātma-nirdvayād ātma-niṣṭhatā
27. jñāna-varjitājñāna-hīna-cit / jñānam asti kiṃ jñātum antaram
28. kiṃ svarūpam ity ātma-darśane / avyayābhavāpūrṇa-cit-sukham
29. bandha-mukty-atītaṃ paraṃ sukham / vindatīha jīvas tu daivikaḥ
30. aham-apetakaṃ nija-vibhānakam / mahad idaṃ tapo ramaṇa vāg iyam
